# 数据特点
起始：速度小，存在过0现象；运动时间短，数据多
后期：速度大，不存在过0现象；运动时间短，数据少
每个来回时间不变

# 流程
- 滤波，消除干扰的过零点
- sign确定分割点：左为正右为负则为一个正、逆转周期
- 求均值：取每一个正/逆方向上中间90%的数据，消除两端的上升下降
- 对每个关节同方向上速度进行拟合，得到函数与参数

# 文件

- friction_para_identification_dir
